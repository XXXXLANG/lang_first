# lang_first
模仿classin的websocket聊天室，提供电脑直播教学功能，javafx构建的桌面客户端
该项目是我的一个练习项目，目前还有很多不足，例如文件上传，头像管理等功能还没有完全实现，退出直播还有bug等等。

以下为功能展示：
一.登录功能，提供账号密码登录、手机验证码登录、界面上的一些按钮功能没有去实现。
